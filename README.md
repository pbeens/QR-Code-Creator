# QR Code Creator

This repository contains a [Jupyter Notebook](https://colab.research.google.com/drive/1rqwtOXbBN4vXeQlj3ftGAuOjbnQWXPVU) designed to run on Google Colab for generating QR codes.

## Run on Colab

Click the link below to open the notebook in Google Colab:

[**Open in Google Colab**](https://colab.research.google.com/github/pbeens/QR-Code-Creator/blob/main/QR_Code_Creator.ipynb)

## Local Usage (VS Code, etc.)

You can also download the notebook to run it locally on your machine using VS Code, JupyterLab, or any other Jupyter environment.

1.  Open the [Colab link](https://colab.research.google.com/github/pbeens/QR-Code-Creator/blob/main/QR_Code_Creator.ipynb).
2.  Click **File** > **Download** > **Download .ipynb**.
3.  Open the downloaded file in VS Code or your preferred editor.
